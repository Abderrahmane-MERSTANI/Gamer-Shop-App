# 🖥️ Elite PC Gaming Store

**Elite PC Gaming Store** is a luxury-themed React Native e-commerce app that lets users browse, view, and purchase high-end gaming laptops.  
It features a sleek **dark neon blue design**, an elegant UI, and smooth interaction for an immersive user experience.

---

## ✨ Features

- 🏠 **Home Page:** Browse a curated list of premium gaming PCs.  
- 💻 **Product Details:** View specs, description, and price with an image preview.  
- 🛒 **Cart System:** Add, remove, or clear products; view total cost dynamically.  
- 💳 **Checkout Simulation:** Confirm your order (mock flow).  
- 🎨 **Luxurious Design:** Neon glow, dark background, modern rounded cards.

---

## ⚙️ Tech Stack

- **React Native (Expo)**
- **React Hooks (useState, useContext)**
- **React Native Components:** `FlatList`, `Modal`, `TouchableOpacity`, `Image`, etc.
- **Custom Context:** Global cart management
